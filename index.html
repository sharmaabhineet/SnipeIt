Developer
----------------
    Abhineet Sharma, pursuing MS in Computer Science at Iowa State University ( abhineet@iastate.edu )

Description
-----------------
    An attempt to build a sniper game with a very basic functionality. All the details are available in the project report file.

How to run
-----------------
    Double click in the Snipeit.exe in the Executable directory. It will start the game.
    Here is the short summary of controls :
        t or T : Enter into Target Mode
        r or R : Reset the game
        Up Arrow : Zoom in
        Down Arrow : Zoom out
        Left Mouse Button : Shoot the bullet

Effort
-----------------
    ~ 15 person hours

Have fun :)
